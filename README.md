# Earnings Call Analyzer

A reusable Claude Code custom command for analyzing earnings call transcripts.

## Requirements

- Claude Code 2.x+

## Installation

Clone the repository:

```bash
git clone https://github.com/YOURNAME/earnings-call-analyzer.git
```

Enter the folder:

```bash
cd earnings-call-analyzer
```

Start Claude Code:

```bash
claude
```

Run:

```text
/earnings-call-analyzer @examples/tsmc-2026-q1.txt
```

## Output

1. Business Performance
2. Future Outlook and Guidance
3. Investment Risks
4. Implications for Investors
